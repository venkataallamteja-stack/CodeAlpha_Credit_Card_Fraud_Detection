# CodeAlpha_Credit_Card_Fraud_Detection
Credit Card Fraud Detection system using Random Forest Classifier. Achieved 95% Precision and 80% Recall on a dataset of 284k+ transactions.

### Project Metrics
| Metric | Score |
| :--- | :--- |
| **Model** | Random Forest Classifier |
| **Precision (Fraud)** | 95% |
| **Recall (Fraud)** | 80% |
| **Accuracy** | 99.9% |

### Key Insights
* **Handling Imbalance:** Successfully managed a highly imbalanced dataset (Class 0: 99.8%, Class 1: 0.2%).
* **Optimization:** Used `StandardScaler` for Time and Amount features to ensure consistent model weighting.
* **Performance:** Focused on high Recall to minimize missed fraudulent transactions.
